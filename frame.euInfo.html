INFO


euDock is a new (cross Platform) OPEN SOURCE (LGPL) javascript emulation of a MacOsX dock bar.

FEATURES
  • Various page align (to screen borders, to html object or to fixed position inside the screen).
     
  • Png alpha and fading support in Interent Explorer.
     
  • Object oriented (progrmmable) icons.
     
  • Customizable venus Hill function.
     
  • MouseClick, MouseMove, MouseUp, MouseDown events.
